Welcome to Hypercore

Hypercore is like a digital ledger or diary where every entry is permanently recorded in the order it was added. Imagine a notebook where you can only add pages at the end—nothing can be erased or changed once written. This unique design ensures that your data is secure, immutable, and always available for review.

In simple terms, Hypercore lets you share large amounts of data quickly and safely across many devices, without relying on a central server. It’s perfect for building peer-to-peer applications that need to work in real time.

Our goal is to make secure, peer-to-peer data sharing as easy as keeping a shared diary among friends.

Understanding Hypercore in Simple Terms

Picture a group chat where every message is written on a permanent note pinned on a board. Once a message is posted, it remains unchanged, ensuring everyone sees the same history of the conversation.

With Hypercore, every new piece of data is like a message added to this board—it is recorded in sequence and cannot be modified later. This means your data is always preserved, secure, and transparent.

What Makes Hypercore Special?

Hypercore is built with several powerful features that work together to create a fast, secure, and efficient system:

  • Sparse Replication: It downloads only the parts of the data you need, saving both time and bandwidth.
  • Realtime Updates: As soon as new data is added, it’s available to everyone instantly.
  • High Performance: A simple, flat file structure keeps operations fast and responsive.
  • Secure Integrity: Using advanced cryptography (like signed Merkle trees), it ensures that your data remains untampered.
  • Modular Design: It’s designed to focus solely on sharing data, making it easy to integrate into various applications.
  • Advanced Tools: Features like on-disk encryption, write-ahead logging, and snapshot views add extra layers of security and convenience.

For an in-depth look at each feature, check out our Features page.

Getting Started with Hypercore

Installing Hypercore is straightforward if you're familiar with Node.js. It is available as an npm package. Simply run the following command in your terminal: 

npm install hypercore

Once installed, Hypercore lets you create a secure and immutable data log for your application—whether it's a real-time chat app, a collaborative tool, or a data streaming service.
